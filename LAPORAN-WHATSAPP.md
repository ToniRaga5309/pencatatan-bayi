📌 *PENGERTIAN*
Sistem web untuk mencatat data kelahiran bayi oleh Puskesmas, yang menjadi dasar pembuatan Akta Kelahiran oleh Dukcapil.

━━━━━━━━━━━━━━━━━━━━━━━━

👥 *PENGGUNA & FITUR*

🔹 *ADMIN DUKCAPIL*
_Berwenang mengelola seluruh sistem_

Menu yang tersedia:
1️⃣ *Dashboard*
   • Melihat total data kelahiran
   • Melihat jumlah data baru
   • Melihat statistik per Puskesmas
   • Grafik data kelahiran

2️⃣ *Data Baru*
   • Daftar data yang belum diunduh
   • Filter berdasarkan tanggal
   • Filter berdasarkan Puskesmas
   • Unduh ke Excel (format siap cetak)

3️⃣ *Register Data*
   • Seluruh data yang sudah tercatat
   • Riwayat pengunduhan
   • Unduh semua data ke Excel

4️⃣ *Kelola User*
   • Tambah operator baru
   • Edit data operator
   • Nonaktifkan akun operator
   • Reset password operator

5️⃣ *Kelola Puskesmas*
   • Tambah Puskesmas baru
   • Edit data Puskesmas

━━━━━━━━━━━━━━━━━━━━━━━━

🔹 *OPERATOR PUSKESMAS*
_Berwenang menginput data kelahiran_

Menu yang tersedia:
1️⃣ *Dashboard*
   • Ringkasan data Puskesmas sendiri
   • Jumlah data yang sudah diinput
   • Data bulan ini

2️⃣ *Input Data*
   Form penginputan:
   • NIK Ibu (16 digit, validasi otomatis)
   • Nama Ibu
   • Nama Ayah
   • Nama Bayi
   • Tanggal Lahir (kalender)
   • Tempat Lahir
   • Jenis Kelamin (L/P)

3️⃣ *Riwayat*
   • Daftar data yang sudah diinput
   • Cari berdasarkan nama/NIK
   • Filter berdasarkan tanggal
   • Lihat detail data

━━━━━━━━━━━━━━━━━━━━━━━━

🔄 *ALUR KERJA LENGKAP*

1️⃣ Puskesmas menerima laporan kelahiran dari masyarakat/Bidan

2️⃣ Operator login ke sistem

3️⃣ Operator menginput data di menu "Input Data"

4️⃣ Data tersimpan otomatis dengan status "Terverifikasi"

5️⃣ Admin Dukcapil login dan melihat data baru

6️⃣ Admin mengunduh data dalam format Excel

7️⃣ Data Excel digunakan untuk pembuatan Akta Kelahiran

8️⃣ Data yang sudah diunduh ditandai sebagai "Sudah Diproses"
